# jsonsearch
Web live search engine for JSON database

[Visit JSONSearch](https://jsonsearch.github.io/)

## Create a database
Acceptable JSON format:
```json
[{
  "title":"title value 1",
  "text":"text value 1"
},{
  "title":"title value 2",
  "text":"text value 2"
}]
```
